# 📱 マネぼう - マーケティング資料

## 🎯 プロジェクト紹介

### 短い説明（30 秒版）

```
💰 マネぼう は、学生がマネ（仮想通貨）を管理し、
先生がマネを付与・回収できるアプリです。

ゲーミフィケーションを通じて、
金銭感覚と経済学習を支援します。

本番環境で稼働中：https://manebou-app.vercel.app
```

### 中程度の説明（1 分版）

```
マネぼう は、学校教育における新しい「マネー教育」プラットフォームです。

【特徴】
✅ 学生が「マネ」（仮想通貨）を管理
✅ 先生が学習成果や良い行動にマネを付与
✅ ランキングでゲーミフィケーション
✅ 取引履歴で金銭感覚を学習

【対象】
- 小中高校の生徒（学習支援）
- 学校の先生（教育ツール）
- 保護者（子どもの金銭管理を遠隔監視）

【メリット】
🎓 金銭感覚の育成
🏆 競争意識の向上
📊 取引履歴での学習記録
💡 経済学習の実践化

本番環境：https://manebou-app.vercel.app
```

```

経済学習を楽しくサポートするアプリです。

【お子さんにとってのメリット】

1️⃣ 金銭感覚が身につく
   - 先生から付与されたマネを管理
   - 取引履歴で「どこにお金を使ったか」を記録
   - 金銭と価値の関係を学習

2️⃣ 経済学習が楽しくなる
   - ランキングで競争意識が刺激される
   - 努力が「マネ」として目に見える報酬に
   - ゲーム感覚で学習継続

3️⃣ 親子で話題にできる
   - 取引履歴を見て「今月はどう使った？」と会話
   - お子さんの金銭管理を一緒に考える機会

【使い方】

1. お子さんがログイン
   → デモメール：student1@school.jp でテスト可能

2. ダッシュボードで残高・履歴を確認
   → シンプルで分かりやすい表示

3. ランキングで成績を確認
   → 他の生徒との比較で刺激

【安心してください】

✅ 安全：SSL/TLS で暗号化通信
✅ シンプル：直感的に使える設計
✅ 透明：親は取引履歴をいつでも確認可能
✅ 教育的：先生が適切に監督

【本番環境】

URL: https://manebou-app.vercel.app
ログイン用メールアドレス（デモ用）: student1@school.jp

ご不明な点は、学校の先生までお気軽にお問い合わせください。
```

## 👨‍🏫 先生向け説明文

### タイトル
```
生徒の金銭感覚を育てる教育ツール
「マネぼう」活用ガイド
```

### 本文

```
「マネぼう」は、生徒の金銭感覚と経済学習を支援する
クラウドベースの教育ツールです。

【先生にとってのメリット】

1️⃣ マネ付与・回収が簡単
   - 生徒を選択して金額を入力するだけ
   - テスト高得点、良い行動、宿題提出などで即座に付与
   - 指導の効果が「見える化」される

2️⃣ クラス全体の管理が楽になる
   - 生徒一覧で全員の残高を一目で確認
   - ダッシュボードでクラス統計を表示
   - 不公正がない公平な管理

3️⃣ 金銭教育が実践的になる
   - 経済の基本（供給と需要）を学習
   - 取引履歴で「お金の流れ」を理解
   - ランキングで競争心を刺激

4️⃣ 保護者との連携がスムーズ
   - 生徒の金銭管理状況を保護者に報告できる
   - 親子で金銭について話題にできる機会

【活用シーン】

📖 テスト対策
   → 高得点の生徒にマネを付与

✍️ 授業参加
   → 積極的に手を挙げた生徒にマネを付与

🏆 競走
   → ランキング上位者に追加ボーナス

🎓 経済学習
   → 実際の取引履歴を題材に「お金について考える」授業

【使い方（簡単 3 ステップ）】

1️⃣ ログイン
   → teacher@school.jp でデモアカウントにアクセス

2️⃣ マネ管理ページを開く
   → /app/teacher/mane-management

3️⃣ 生徒を選択して、金額・理由を入力し付与
   → リアルタイムで生徒の残高が更新される

【サポート】

- 完全なマニュアル付属
- 24/7 本番環境稼働
- API で他システムとの連携も可能

【本番環境】

URL: https://manebou-app.vercel.app
ログイン用メールアドレス（デモ用）: teacher@school.jp

詳しくは、README.md をご覧ください。
```
