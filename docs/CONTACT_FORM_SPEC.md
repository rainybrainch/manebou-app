# お問い合わせフォーム仕様

## 概要

マネぼうへのお問い合わせを受け付けるフォーム仕様書です。

**実装予定**: Phase 11b（v1.0 リリース前）
**URL**: `/contact`
**実装方式**: React Server Action + sendgrid API

---

## フォーム項目

### 必須項目

| 項目 | 型 | バリデーション | 説明 |
|------|-----|---------|------|
| **名前** | text | 1-50字 | お客様の名前 |
| **メールアドレス** | email | RFC 5322準拠 | 返信用メールアドレス |
| **学校名** | text | 1-100字 | 学校・団体名 |
| **立場** | select | 保護者/先生/学校管理者/その他 | ユーザーのポジション |
| **お問い合わせ内容** | textarea | 10-2000字 | 詳細な質問・要望 |

### オプション項目

| 項目 | 型 | 説明 |
|------|-----|------|
| **電話番号** | tel | 連絡がつきやすい番号 |
| **生徒数** | select | 導入検討時に参考 |
| **予算** | select | 導入規模の参考 |

---

## バリデーションルール

```typescript
interface ContactFormData {
  name: string           // 必須・1-50字
  email: string          // 必須・メール形式
  school: string         // 必須・1-100字
  position: 'parent' | 'teacher' | 'admin' | 'other'  // 必須
  message: string        // 必須・10-2000字
  phone?: string         // オプション
  studentCount?: 'under50' | '50-200' | '200-500' | 'over500'  // オプション
  budget?: 'under100k' | '100-500k' | 'over500k'  // オプション
}
```

### エラーメッセージ

| 項目 | エラーメッセージ |
|------|------|
| 名前 | 「名前は1-50字で入力してください」 |
| メール | 「有効なメールアドレスを入力してください」 |
| 学校 | 「学校名は1-100字で入力してください」 |
| 立場 | 「立場を選択してください」 |
| 内容 | 「お問い合わせ内容は10-2000字で入力してください」 |

## 送信後の処理

### 成功時

1. **ユーザー画面**:
   - 成功メッセージ表示: 「ご送信ありがとうございました。24時間以内にご返信いたします。」
   - 3秒後に自動でホームに遷移

2. **バックエンド**:
   - データベースに保存
   - SendGrid でメール送信（管理者 + ユーザー）
   - 通知 Slack に投稿（管理者向け）

### エラー時

1. **ユーザー画面**:
   - エラーメッセージ表示: 「送信に失敗しました。お手数ですが、もう一度お試しください。」
   - フォーム内容は保持

2. **バックエンド**:
   - エラーログに記録
   - 管理者に Slack 通知

## レート制限

- **IP単位**: 1時間に10件まで
- **メール単位**: 1日に3件まで
- **バースト**: 1分に1件まで

エラーメッセージ: 「お問い合わせは一定期間に送信できません。お手数ですが、しばらく経ってからお試しください。」

## 保存先

**データベース**: `contacts` テーブル

```sql
CREATE TABLE contacts (
  id: string (UUID)
  name: string
  email: string
  school: string
  position: string
  message: string
  phone: string | null
  student_count: string | null
  budget: string | null
  ip_address: string
  user_agent: string
  created_at: timestamp
  updated_at: timestamp
  status: 'new' | 'replying' | 'closed'
)
```

## SLA 定義

| 項目 | SLA |
|-----|-----|

| 初回返信 | 24時間以内 |
| 最終返信 | 3営業日以内 |
| 平日営業時間 | 09:00-18:00 (JST) |
