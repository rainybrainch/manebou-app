# マネぼう ユーザーガイド

## 📖 はじめに

マネぼうは、学生がマネ（仮想通貨）を管理し、金銭感覚と経済学習を身につけられるアプリです。  
先生が学習成果に応じてマネを付与し、生徒がランキングで競い合うゲーミフィケーション型の教育アプリです。

---

## 🎓 学生向けガイド

### ログイン

1. [https://manebou-app.vercel.app](https://manebou-app.vercel.app) にアクセス
2. メールアドレスを入力（例: `student1@school.jp`）
3. 「ログイン」ボタンをクリック
4. **ダッシュボード** が表示される

### ダッシュボード（見方）

| 項目 | 説明 |
|------|------|
| **現在のマネ** | 今持っているマネの残高 |
| **今週の増減** | 先週比でどれだけ増えたか |
| **クイックアクション** | よく使う機能へのショートカット |
| **最近の取引** | 先生からもらったマネの記録 |

### マネ履歴を確認する

1. ダッシュボードの「📜 マネ履歴を見る」をクリック
2. **すべての取引履歴** が表示される

**機能:**
- **日時順**: 新しい順に表示
- **金額順**: 多い順に表示
- **統計情報**: 現在残高・今日の増減・総獲得・総減算

### ランキングを確認する

1. ダッシュボードの「🏆 ランキングを確認」をクリック
2. **学校全体のランキング** が表示される

**メダル表示:**
- 🥇 1位（黄金）
- 🥈 2位（銀）
- 🥉 3位（銅）
- 4位以下：リスト表示

## 👨‍🏫 先生向けガイド

### ログイン

1. [https://manebou-app.vercel.app](https://manebou-app.vercel.app) にアクセス
2. メールアドレスを入力（例: `teacher@school.jp`）
3. 「ログイン」ボタンをクリック
4. **先生用ダッシュボード** が表示される

### ダッシュボード（見方）

| カード | 説明 |
|--------|------|
| **学校** | 学校名 |
| **生徒数** | クラス内の生徒数 |
| **クラス数** | 担当クラス数 |
| **今日配布** | 本日配布したマネ（目標比） |
| **今週配布量** | 先週比での配布量 |

### 生徒一覧を確認する

1. 先生用ダッシュボードの「👥 生徒一覧」をクリック
2. **クラス内の全生徒** がテーブルで表示される

**検索方法:**
- 名前またはメールアドレスで検索
- リアルタイムで結果が絞られる

### マネを付与する

1. 先生用ダッシュボードの「💸 マネ付与・回収」をクリック
2. **生徒を選択**
3. **操作タイプ**: ✅ 付与する を選択
4. **金額**: クイックボタンで選択（50/100/200/500）
5. **理由**: 選択
   - 授業参加
   - 宿題提出
   - クイズ正解
   - 良い行動
   - その他
6. **メモ**: 任意で記入（例: 積極的な発表をしていました）
7. **実行** をクリック

### マネを回収する

1. 先生用ダッシュボードの「💸 マネ付与・回収」をクリック
2. **生徒を選択**
3. **操作タイプ**: ❌ 減算する を選択
4. **金額**: クイックボタンで選択
5. **理由**: 選択
6. **実行** をクリック

## ❓ よくある質問

### Q: デモユーザーで永続的にデータが保存されるか？
**A:** いいえ。デモユーザーのデータはブラウザの localStorage に保存されます。ブラウザをクリアするとデータは消えます。本番運用では、データベースに保存されます。

### Q: パスワードは何か？
**A:** デモユーザーではパスワードは不要です。本番運用では認証システムが導入されます。

### Q: ログアウトするには？
**A:** ページを離れるか、ブラウザを閉じてください。次回訪問時には再度ログインが必要です。

### Q: スマートフォンで使えるか？
**A:** はい。マネぼうはレスポンシブ設計で、スマートフォンでも快適に利用できます。
