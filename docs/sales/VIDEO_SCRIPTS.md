# マネぼう デモビデオスクリプト

## シリーズ概要

マネぼうの魅力を伝える 3 種類のビデオスクリプト

1. **3 分版**: 学校説明会・営業説明会向け（コンパクト）
2. **10 分版**: 教育委員会・研修会向け（詳細）
3. **30 分版**: 先生向け操作研修・デモ動画（実操作中心）
