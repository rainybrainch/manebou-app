## 2. マネぼうのソリューション

### 2.1 仮想通貨管理システム

先生が生徒にマネ（仮想通貨）を付与 → 生徒がマネ残高を管理

**効果**:
- 学習成果が即座に反映される
- 生徒が自分の進捗を数値で確認できる
- 金銭感覚が自然に身につく

### 2.2 ランキング機能

全生徒のマネ保有額をリアルタイム表示

**効果**:
- 学生同士が競い合い、学習意欲が向上
- 上位ランカーのモチベーションが維持される
- 全校で一体感が生まれる

### 2.3 マネ履歴管理

生徒が過去のすべての取引を確認

**効果**:
- 「どのような行動でマネが増えたのか」が視覚化
- 金銭管理スキルが実践的に身につく
- 座学の経済知識が「生きた知識」に
