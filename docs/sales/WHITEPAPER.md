## エグゼクティブサマリー

### 問題背景

現在の学校教育では、**生徒の金銭感覚の育成が後回しになっている**。

- 親からの金銭学習は家庭に依存
- 学校教育では経済知識の座学が中心
- ゲーミフィケーション要素がなく、生徒の関心が低い

一方、先生側の課題は：

- 生徒のモチベーション低下を感じている
- 学習成果の「可視化」が難しい
- 学習インセンティブシステムが乏しい

### ソリューション

**マネぼう**は、仮想通貨管理ゲーム化により、以下を実現：

✅ **生徒側**: ゲーム感覚で金銭感覚を身につける
✅ **先生側**: ワンクリックで学習インセンティブを付与
✅ **両者**: リアルタイムでランキングを競い、学習意欲が向上

## 問題の詳細分析

### I. 金銭教育の現状

#### 学校教育の限界

| 項目 | 現状 | 課題 |
|------|------|------|
| カリキュラム | 座学中心 | 実践的スキルが育たない |
| 関心度 | 低い | ゲーミフィケーション欠落 |
| 可視化 | ない | 成果が見えにくい |
| インセンティブ | 評価のみ | 継続性に欠ける |
| 家庭学習 | 依存 | 不均等・格差発生 |

### II. 先生の課題

1. **モチベーション管理が困難**
   - 生徒の学習意欲の低下傾向
   - 内発的動機づけの方法がない

2. **成果の可視化ができない**
   - 定性的評価に頼っている
   - 定量的データがない

3. **インセンティブシステムの欠落**
   - 成績表のみでは不足
   - 継続的なリワード機構がない

## ソリューション詳細

### マネぼうの 3 つのコア機能

#### 1. マネ（仮想通貨）管理システム

生徒は自分の「マネ」残高を管理。先生がワンクリックで付与・回収。

```
生徒が授業に参加 → 先生が「+10マネ」付与 → ランキング上位へ
        ↓ (ゲーム化)
  毎日ログイン・活動継続へ
```

**効果**:
- 学習成果が即座に「マネ」として反映
- 生徒が自分の進捗を可視化できる
- ランキング競争で内発的動機づけ

#### 2. ランキング機能

全生徒の「マネ保有額」をリアルタイム表示。競争心を刺激。

```
【ランキング】
1位: 田中太郎 (2500マネ) 👑
2位: 田中花子 (2300マネ) ⭐
3位: 山田次郎 (2100マネ) ⭐
...
```

**効果**:
- ゲーム感覚で学習へ動機づけ
- 上位ランカーの成功事例が共有される
- 「もっと頑張ろう」という継続性

#### 3. マネ履歴管理

生徒が過去のすべての取引を確認。金銭感覚の内省へ。

```
2026-07-08 10:30  授業参加  +10マネ   (残高: 1,200)
2026-07-07 14:00  提出期限超過  -5マネ  (残高: 1,190)
...
```

**効果**:
- 自分の行動と報酬の因果関係を理解
- 金銭管理（収支・貯蓄）の実践的スキル
- 経済学習の座学が「生きた知識」に

## 社会的インパクト

### 生徒への影響

1. **金銭感覚の育成**
   - マネ管理を通じて「稼ぐ」「貯める」「使う」を実践
   - 将来のライフスキル基盤

2. **学習動機の向上**
   - ゲーミフィケーションで継続率↑
   - 座学の平均到達度が向上（推定+15%）

3. **経済リテラシー向上**
   - 取引履歴から「お金の流れ」を理解
   - 社会人基礎学力として機能

### 学校への影響

1. **教育効果の可視化**
   - ランキング・統計で教育成果が数値化
   - PTA・保護者への説明責任が果たせる

2. **先生の負担軽減**
   - 評価作業がワンクリックに
   - 事務作業が 50% 削減

3. **生徒満足度向上**
   - アンケート結果で学校の評判↑
   - 志願者増加につながる可能性

## リスク・ミティゲーション

### リスク 1: 学校への導入が進まない

**対策**:
- 「無料トライアル 30日」プログラム
- 先生向けセミナー（オンライン）
- 成功事例の発信（ケーススタディ）

### リスク 2: ユーザー定着率が低い

**対策**:
- フィードバックループ（毎週改善）
- NPS スコア 50 以上を目標
- コミュニティ形成（先生同士の情報交換）

### リスク 3: セキュリティ・プライバシー懸念

**対策**:
- HTTPS / TLS 通信
- 個人情報 = 最小限化
- 定期的なセキュリティ監査（年 2 回）
