## 1. マネぼう君 キャラクター仕様

### 1.1 基本情報

| 項目 | 内容 |
|-----|------|
| **名前** | マネぼう君 |
| **性別** | 中性（子ども向けで親しみやすく） |
| **種族** | マネの国に住む魔法のマネコイン生命体 |
| **年齢** | 永遠の 8 歳（成長しない） |
| **住まい** | マネの国（ゲーム世界） |

### 1.2 役割・立場

```
ユーザーの視点から：

【子ども向け】
- 友人・遊び相手
- 一緒に学ぶ仲間
- 励まし・応援者

【親向け】
- 子どもの学習パートナー
- 信頼できるガイド
- 親子の会話のきっかけ

【先生向け】
- 授業サポートツール
- 生徒のモチベーション向上
- 金銭教育のキャラクター化
```

### 1.3 性格・特徴

| 特性 | 説明 |
|-----|------|
| **明るさ** | いつもニコニコ。失敗も学習と捉える前向きさ |
| **やさしさ** | 子ども目線で話す。難しい説明はしない |
| **おしゃべり** | 時々「マネの知識」を教えてくれる |
| **サポート精神** | 困っている人を助けるのが好き |
| **冒険心** | マネの国の秘密を一緒に探検したい |

### 1.4 ビジュアルデザイン

#### ベース形状
```
大きな円形（コイン）
↓ 金色
↓ 温かみのある光沢
↓ 3D 的な立体感
```

#### 顔パーツ
- **目**: 黒い丸（大きめ、つぶらで可愛らしい）
- **眉毛**: 細く、感情で動く（角度で表現）
- **口**: シンプルな曲線（表情差分で変化）
- **鼻**: 小さい点（シンプルさを重視）

#### 体パーツ
- **腕**: 短く、丸い手（手袋を被った感じ）
- **足**: 短く、ぷくぷくした脚
- **王冠**: 小さく頭に乗っている（マネの王様イメージ）
- **バッジ**: 右胸に「💰」マークがついている

#### 色
- **本体**: 金色（`#f59e0b`） + ハイライト白（`#ffffff`）
- **目**: 黒色（`#1f2937`）
- **王冠**: 紺色（`#1e3a8a`） + 金色アクセント
- **バッジ**: 緑色（`#10b981`）背景

#### スタイル参考
- **雰囲気**: 温かみのある家庭用ゲーム風 3D キャラクター
- **親しみやすさ**: あたたかく優しい表情・ポーズ
- **シンプルさ**: 線画版・イラスト版で再現可能なレベル

### 1.5 表情差分（全 5 パターン）

#### 1️⃣ デフォルト（微笑み）
```
・用途: ホームページ、通常時、待機画面
・表情: 目が半月形、口が「∧」型の微笑み
・イメージ: やさしく、親しみやすい
・説明: 「一緒にやろうね」という感じ

[SVG描画例]
目: arc（半月）
口: path（〜型の曲線）
眉毛: 水平（リラックス）
```

#### 2️⃣ ガッツポーズ
```
・用途: ランキング 1 位、クイズ正解、達成時
・表情: 目が〇（開いて元気）、口が大きな笑み
・ポーズ: 両手を上げて「やった！」
・イメージ: 喜び・興奮・達成感
・説明: 「成功した！」「おめでとう！」

[SVG描画例]
目: 〇（大きく開く）
口: path（喜びの笑み）
眉毛: /\ 形（上昇、喜び）
腕: 上向き（ポーズ）
```

#### 3️⃣ 考え中（疑問）
```
・用途: チュートリアル、ヒント提示、説明時
・表情: 目が「？」、口が「ー」
・ポーズ: あごに手を当てる
・イメージ: 考える、質問する、教える
・説明: 「これはどうかな？」「考えてみて」

[SVG描画例]
目: 「？」記号
口: 直線（真顔）
眉毛: ハの字（思考）
腕: あご手（考える仕草）
```

#### 4️⃣ 応援（励まし）
```
・用途: 失敗時、励ましメッセージ、応援画面
・表情: 目が✨、口が「応援」型の笑み
・ポーズ: 両手を胸の前で合わせる（応援）
・イメージ: 励まし、サポート、応援
・説明: 「頑張ってね！」「応援してる」

[SVG描画例]
目: ✨（光のエフェクト）
口: 〜型の優しい笑み
眉毛: / \ （上昇、応援）
腕: 胸前で合わせる
```

#### 5️⃣ 驚き（デバイス通知）
```
・用途: 新機能紹介、重要な知らせ、イベント通知
・表情: 目が「○」（大きく驚く）、口が「O」（びっくり）
・ポーズ: 両手を上げて驚く
・イメージ: 驚き、新しい発見、ニュース
・説明: 「わあ！」「新しいのが来たよ！」

[SVG描画例]
目: 〇（大きく開く）
口: O 型（驚き）
眉毛: Λ 形（上昇、驚き）
頭上: 💡（発見）
```

#### 6️⃣ 解説（先生・教育）
```
・用途: 教材ページ、授業、チュートリアル、ガイド説明
・表情: 目が「◉」（優しく見守る）、口が「説明する」型の笑み
・ポーズ: 本を持つ or 手を広げて説明する仕草
・イメージ: 教える、サポート、優しさ、知識共有
・説明: 「一緒に学ぼう」「ここが大切だよ」「わかった？」

[SVG描画例]
目: ◉（やさしく見守る）
口: 〜型の教える笑み
眉毛: / \ （上昇、優しい）
腕: 本を持つ or 説明するジェスチャー
本: 左手に「本」を持っている場合
```

### 1.6 アニメーション・動き

#### ふわふわ浮き上がり
```
・パターン: y 軸方向に -10px～+10px で浮遊
・タイミング: 無限ループ（4 秒周期）
・用途: 待機画面、Hero、ホームページ
```

#### 点滅（応援・通知時）
```
・パターン: opacity 0.6～1.0 で点滅
・タイミング: 0.5 秒周期
・用途: 新機能通知、アラート
```

#### 回転（ローディング）
```
・パターン: 360° 回転
・タイミング: 2 秒周期
・用途: データ読み込み中、計算中
```

## 2. ブランドカラーシステム

### 2.1 メインカラーパレット

| 役割 | 色名 | HEX | RGB | Tailwind | 用途 |
|-----|------|-----|-----|----------|------|
| **プライマリ** | 紺（Navy Blue） | `#1e3a8a` | 30, 58, 138 | `blue-900` | ヘッダー、本文、重要要素、信頼感 |
| **セカンダリ** | 金（Amber） | `#f59e0b` | 245, 158, 11 | `amber-500` | マネぼう君、CTA、アクセント |
| **アクセント** | 緑（Emerald） | `#10b981` | 16, 185, 129 | `emerald-500` | 成功表示、チェック、成長 |
| **背景** | クリーム（Amber Light） | `#fffbeb` | 255, 251, 235 | `amber-50` | ページ背景、カード背景 |

### 2.2 拡張カラーパレット

#### グレースケール（テキスト・境界線）
```
濃いグレー: #1f2937（gray-800）- 本文
中程度グレー: #6b7280（gray-500）- サブテキスト
淡いグレー: #e5e7eb（gray-200）- 境界線・分割線
```

#### ステータスカラー
```
成功: #10b981（緑・Emerald）
警告: #f97316（オレンジ・Orange）
エラー: #dc2626（赤・Red）
情報: #3b82f6（青・Blue）
```

### 2.3 グラデーション

#### Hero セクション背景
```css
background: linear-gradient(135deg, 
  #e0e7ff 0%,      /* 紺の淡さ */
  #fef3c7 50%,     /* クリーム */
  #d1fae5 100%     /* 緑の淡さ */
);
```

#### ボタン・CTA
```css
background: linear-gradient(135deg,
  #1e3a8a 0%,      /* 紺 */
  #f59e0b 100%     /* 金 */
);
```

#### マネぼう君・キャラ周辺
```css
background: radial-gradient(circle,
  #fef3c7 0%,      /* クリーム（中心） */
  #fed7aa 100%     /* 金の淡さ（周辺） */
);
```

### 2.4 色の使い分けルール

#### 紺（`#1e3a8a`） - 信頼・安定
```
✓ ページ背景（薄く）
✓ ヘッダー・フッター
✓ 本文テキスト
✓ ボタンベース
✓ 見出し（h1-h3）
✗ 子ども向けキャラは避ける（固い印象）
```

#### 金（`#f59e0b`） - マネ・親しみやすさ
```
✓ マネぼう君（メインカラー）
✓ CTA ボタン（目立たせる）
✓ ハイライト・アクセント
✓ 成功・報酬表示
✗ 大面積での背景（疲れやすい）
```

#### 緑（`#10b981`） - 成長・安心
```
✓ チェックマーク
✓ 成功表示
✓ プログレスバー
✓ 親しみやすさ
✗ テキストの主要色（読みにくい）
```

#### クリーム（`#fffbeb`） - 親しみやすさ
```
✓ 背景色（白より温かい）
✓ カード背景
✓ セクション背景
✗ テキスト色（読みにくい）
```

## 3. Hero 右側 キャラクター表示エリア設計

### 3.1 レイアウト構成

```
┌────────────────────────────────────────────┐
│                                            │
│            【光の玉エリア】                │
│         ✨ ✨ ✨（装飾）                   │
│                                            │
│        ┌──────────────────┐               │
│        │                  │               │
│        │  マネぼう君     │               │
│        │  イラスト      │               │
│        │                  │               │
│        │  400×400px      │               │
│        │                  │               │
│        └──────────────────┘               │
│                                            │
│         💫 グロー効果（金色）             │
│                                            │
└────────────────────────────────────────────┘
```

### 3.2 デバイスごとの表示

#### デスクトップ（1024px 以上）
```
・表示: フル表示（400×400px）
・位置: Hero 右側に堂々と配置
・周辺装飾: 光の玉 3-4 個、グロー効果
・説明: マネぼう君の全身がしっかり見える
```

#### タブレット（768px～1024px）
```
・表示: 300×300px に縮小
・位置: Hero 右側に配置
・周辺装飾: グロー効果のみ
・説明: バランスの良いサイズ
```

#### スマートフォン（375px～768px）
```
・表示: 非表示 or 200×200px に縮小
・位置: テキストの下に配置（full-width）
・周辺装飾: シンプル
・説明: モバイルではテキスト優先
```

### 3.3 HTML 構造例

```html
<!-- Hero 右側のキャラクター領域 -->
<div class="hidden lg:flex items-center justify-center">
  <!-- 背景グラデーション -->
  <div class="absolute inset-0 bg-gradient-to-br from-amber-50 to-amber-100 rounded-3xl"></div>
  
  <!-- 光の玉装飾 -->
  <div class="absolute top-10 right-10 w-16 h-16 bg-yellow-300 rounded-full opacity-40 blur-2xl"></div>
  <div class="absolute bottom-20 left-10 w-12 h-12 bg-green-300 rounded-full opacity-30 blur-2xl"></div>
  
  <!-- マネぼう君 SVG or IMG -->
  <div class="relative z-10 animate-bounce">
    <img 
      src="/images/manebou-kun-default.svg" 
      alt="マネぼう君" 
      class="w-96 h-96 drop-shadow-lg hover:drop-shadow-2xl transition-shadow"
    />
  </div>
</div>
```

### 3.4 CSS アニメーション

```css

  0%, 100% { transform: translateY(0px); }
  50% { transform: translateY(-20px); }

}

  filter: drop-shadow(0 0 20px rgba(245, 158, 11, 0.4));
```
