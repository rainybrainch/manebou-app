# マネぼう 既知の課題

## 📋 既知の課題一覧

このドキュメントは、マネぼう v1.0.0 の既知の課題と制限事項を記載しています。

---

## 🟡 現在のバージョン（v1.0.0）

### 認証・セキュリティ

**【重要】デモユーザーのみ対応**
- 現在はデモユーザーで固定認証
- 実運用には認証システムが必要
- **対応予定**: v1.1.0 で OAuth/OIDC 実装予定

| 課題 | 影響度 | 対応予定 |
|------|--------|---------|
| パスワード認証がない | 🔴 高 | v1.1.0 |
| ユーザー登録機能がない | 🔴 高 | v1.1.0 |
| 暗号化パスワード保存がない | 🔴 高 | v1.1.0 |
| セッション管理が簡易的 | 🟡 中 | v1.1.0 |

### データベース

**【重要】ローカルストレージのみ**
- データは localStorage（ブラウザ内）に保存
- ブラウザをクリアするとすべてのデータが失われる
- 複数デバイス間での同期ができない
- **対応予定**: v1.1.0 でサーバーサイドDB実装予定

| 課題 | 影響度 | 対応予定 |
|------|--------|---------|
| データの永続化がない | 🔴 高 | v1.1.0 |
| 複数デバイス同期ができない | 🔴 高 | v1.1.0 |
| バックアップ機能がない | 🟡 中 | v1.1.0 |

### 機能制限

**【重要】デモ環境固定**
- デモユーザーのデータのみ使用可能
- 実際のユーザーデータを追加できない
- **対応予定**: v1.1.0 で本物の管理画面実装予定

| 課題 | 影響度 | 対応予定 |
|------|--------|---------|
| ユーザー追加ができない | 🔴 高 | v1.1.0 |
| クラス管理ができない | 🔴 高 | v1.1.0 |
| 学校情報の変更ができない | 🟡 中 | v1.1.0 |

## 📅 今後のロードマップ

### v1.1.0 予定（2026年8月）

優先度の高い順：

**MUST（必須）**
- 本物の認証システム（OAuth/OIDC）
- データベース実装（PostgreSQL）
- ユーザー管理機能

**SHOULD（望ましい）**
- メール通知機能
- 詳細な統計レポート
- 管理画面の充実

**COULD（あると良い）**
- ダークモード
- 多言語対応
- モバイルアプリ化
