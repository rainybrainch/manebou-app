# マネぼう 先生向け導入手順

## 📚 学校での実運用開始手順

### 🎯 対象
学校の先生向け導入ガイド

### ⏱️ 導入期間
初回セットアップ: 1 時間  
クラス運用開始: 当日

---

## 📋 導入前に確認

- [ ] 学校から承認を得ている
- [ ] インターネット環境がある
- [ ] PC またはタブレットを用意している
- [ ] 生徒のメールアドレスリストがある（運用時）

---

## 🚀 セットアップ手順

### 1️⃣ ユーザー登録申請
**管理者に以下を連絡:**

```
【マネぼう登録申請】

氏名: [先生名]
メール: [school.jp]
所属: [学校名]・[クラス名]
生徒数: [数]
```

### 2️⃣ ログイン開始（管理者から通知待ち）

**管理者が用意したメールアドレス・パスワードでログイン:**

https://manebou-app.vercel.app

### 3️⃣ ダッシュボード確認

**先生用ダッシュボードが表示される:**

- 学校情報
- 生徒数・クラス数
- 今日配布したマネ
- クイックアクション

### 4️⃣ 生徒一覧を確認

**「👥 生徒一覧」から:**

- クラスの全生徒が表示される
- 各生徒のマネ残高が表示される

### 5️⃣ マネを付与してみる

**「💸 マネ付与・回収」から:**

1. 生徒を選択
2. 付与額を設定（クイックボタンで 100 を選択）
3. 理由を選択（「授業参加」など）
4. メモを入力（任意）
5. 「実行」をクリック

✅ 成功メッセージが表示されたら成功！

---

## 📖 日々の運用

### 毎日の作業（5分）

```
09:00  授業開始 → マネ付与の準備
12:00  授業終了後 → 参加生徒に 100 マネ付与
15:30  課題提出時 → 提出生徒に 50 マネ付与
```

### 週 1 回の確認（10分）

- ランキングを確認
- 生徒のモチベーション変化を観察

### 月 1 回の報告（15分）

- 学校管理者にマネぼうの利用状況を報告
- 生徒の改善点・学習成果を記録

---

## ⚙️ 設定

### パスワード変更（推奨）
初回ログイン時に管理者から通知されたパスワードを変更してください。

### 時間帯設定（将来実装予定）
v1.1.0 以降で、学校の授業時間に合わせた通知機能が実装予定です。

## 🎓 生徒への説明（推奨）

クラスに導入時、生徒に以下を説明してください：

```
「マネぼうは、学習成果に応じて先生がマネを付与します。
このマネを使ってランキングで競い合い、
金銭感覚と経済学習を身につけるアプリです。

不正行為はマネが没収される場合があります。」
```
