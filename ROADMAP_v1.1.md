## 📊 MoSCoW 優先度分析

### 🔴 MUST（必須・実装必須）

これがなければ本番運用不可。

#### 1. 認証システム実装（8 点数日）
- **内容**: OAuth / OIDC による実ユーザー認証
- **実装**: Google 認証 + カスタム認証
- **テスト**: ログイン / ログアウト / セッション管理
- **デリバリー**: 2026-07-25

#### 2. データベース実装（10 点数日）
- **内容**: PostgreSQL + Prisma ORM
- **テーブル**: Users, Schools, Classes, Transactions
- **テスト**: CRUD 操作・トランザクション・ロックアウト
- **デリバリー**: 2026-07-31

#### 3. ユーザー管理画面（5 点数日）
- **内容**: 管理者向けユーザー追加・編集・削除
- **テスト**: ロールベースアクセス制御（RBAC）
- **デリバリー**: 2026-08-08

#### 4. セキュリティ対応（3 点数日）
- **内容**: HTTPS / CSRF 対策 / XSS 対策
- **テスト**: セキュリティテスト
- **デリバリー**: 2026-08-10

**小計**: 26 点数日（実装者 1人で 4 週間）

---

### 🟡 SHOULD（望ましい・優先度高）

実装すれば本番体験が大幅に向上。

#### 1. メール通知機能（4 点数日）
- **内容**: SendGrid 統合・メール配信
- **機能**: マネ付与時・ランキング更新時の通知
- **テスト**: メール送信・テンプレート確認
- **優先度**: HIGH
- **デリバリー**: 2026-08-15

#### 2. 詳細な統計レポート（6 点数日）
- **内容**: グラフ・チャート表示・CSV エクスポート
- **機能**: 先生向けに月別統計・生徒別統計
- **テスト**: データ正確性・グラフ表示
- **優先度**: HIGH
- **デリバリー**: 2026-08-20

#### 3. 管理画面充実（5 点数日）
- **内容**: 学校・クラス・ユーザー管理 UI
- **機能**: 一括操作・フィルター・ソート
- **テスト**: UI テスト・操作性確認
- **優先度**: MEDIUM
- **デリバリー**: 2026-08-25

**小計**: 15 点数日

---

### 💙 COULD（あると良い・優先度低）

実装すればより良いが、v1.1.0 では見送り可能。

#### 1. ダークモード対応（3 点数日）
- **内容**: UI テーマ切り替え
- **実装**: Tailwind CSS theme 対応
- **優先度**: LOW
- **予定**: v1.2.0 以降

#### 2. 多言語対応（5 点数日）
- **内容**: i18n 実装・英語・中文対応
- **優先度**: LOW
- **予定**: v1.2.0 以降

#### 3. プッシュ通知（4 点数日）
- **内容**: Web Push API 統合
- **優先度**: MEDIUM
- **予定**: v1.1.1 で検討

#### 4. モバイルアプリ化（15 点数日）
- **内容**: React Native 実装
- **優先度**: LOW
- **予定**: v2.0.0 以降

**小計**: 27 点数日（見送り推奨）

---

### ❌ WON'T（やらない・v1.1.0 では対象外）

| 機能 | 理由 | 検討時期 |
|------|------|---------|
| AI チャットボット | スコープ大 | v2.0.0 |
| 暗号資産連携 | 複雑・規制対応 | v3.0.0 |
| 3D ゲーム化 | 高コスト・ROI低 | 見送り |
| ブロックチェーン対応 | 過度な実装 | 見送り |
